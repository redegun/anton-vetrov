SEO-продвижение можно делать самостоятельно. Это не требует бюджета на агентство — только время и системный подход. В этой статье — пошаговый план, который подходит для любого сайта: от блога до интернет-магазина.

## Шаг 1: Поставьте цели

Прежде чем что-то оптимизировать, определите, чего хотите:

- **Трафик** — увеличить посещаемость на X% за 6 месяцев
- **Позиции** — попасть в топ-10 по конкретным запросам
- **Конверсии** — получать больше заявок / продаж из органики
- **Видимость** — охватить максимум запросов в своей нише

Без конкретной цели вы не сможете оценить результат.

## Шаг 2: Подключите аналитику

Без данных — нет SEO. Подключите:

- **Яндекс.Метрика** — основной инструмент аналитики для Рунета
- **Яндекс.Вебмастер** — показывает, как Яндекс видит ваш сайт
- **Google Search Console** — аналог для Google
- **Google Analytics 4** — дополнительная аналитика

**Что настроить сразу:**
- Цели в Метрике (заявка, покупка, звонок)
- Привязку домена в Вебмастере
- Подтверждение права собственности в Search Console

## Шаг 3: Проведите технический аудит

Проверьте сайт на технические ошибки. Это фундамент — без него остальная работа бессмысленна.

**Чек-лист:**

- [ ] Сайт работает по HTTPS
- [ ] Скорость загрузки < 3 секунд (PageSpeed Insights)
- [ ] Мобильная версия работает корректно
- [ ] Нет дублей страниц (www / без www, http / https)
- [ ] robots.txt настроен правильно
- [ ] Sitemap.xml создан и отправлен в поисковики
- [ ] Нет битых ссылок (404 ошибки)
- [ ] Изображения оптимизированы (WebP, lazy loading)
- [ ] Нет редиректных цепочек

Подробнее: [SEO-аудит сайта: что проверять](/blog/seo-audit-sajta/).

## Шаг 4: Соберите семантическое ядро

Семантическое ядро — список ключевых слов, по которым вы хотите ранжироваться.

**Быстрый способ для начинающих:**

1. Откройте Яндекс.Wordstat
2. Введите главный запрос вашей ниши
3. Выпишите все релевантные вариации с частотностью
4. Проверьте конкурентов через Keys.so или Serpstat
5. Сгруппируйте запросы по страницам

**Для нового сайта** — начинайте с низкочастотных запросов (до 100 показов в месяц). По ним проще выйти в топ.

Подробный гайд: [Как подобрать ключевые слова для сайта](/blog/kak-podobrat-kljuchevye-slova/).

## Шаг 5: Оптимизируйте существующие страницы

Прежде чем создавать новый контент, доработайте то, что уже есть:

**Мета-теги:**
- Title — до 60 символов, содержит основной ключ
- Description — до 160 символов, мотивирует кликнуть
- H1 — один на страницу, содержит ключ

**Контент:**
- Добавьте недостающие ответы на подвопросы
- Вставьте таблицы, списки, изображения
- Обновите устаревшую информацию
- Убедитесь, что текст отвечает на запрос пользователя

**Внутренние ссылки:**
- Свяжите тематически близкие страницы
- Используйте анкоры с ключевыми словами (но не спамьте)

Подробнее: [Внутренняя оптимизация сайта: чек-лист](/blog/vnutrennjaja-optimizacija-sajta/).

## Шаг 6: Создайте контент-план

Регулярная публикация контента — двигатель SEO. Составьте план на 1–3 месяца вперёд:

1. Возьмите кластеры из семантического ядра
2. Приоритизируйте по потенциалу трафика и конкуренции
3. Запланируйте 2–4 статьи в месяц
4. Каждая статья — под конкретный кластер ключевых слов

**Формула хорошей SEO-статьи:**
- Отвечает на поисковый запрос
- Лучше, чем топ-3 конкурентов в выдаче
- Содержит уникальный контент (опыт, данные, примеры)
- Структурирована (H2, H3, списки, таблицы)

Как писать тексты для SEO: [SEO-оптимизация текста: как писать для поисковиков](/blog/seo-optimizacija-teksta/).

## Шаг 7: Наращивайте ссылочную массу

Ссылки с других сайтов — сигнал авторитетности для поисковиков. Без них сложно выйти в топ по конкурентным запросам.

**Белые методы получения ссылок:**

- Регистрация в каталогах и справочниках (Яндекс.Бизнес, 2ГИС)
- Гостевые статьи на тематических ресурсах
- Комментарии и ответы на форумах (с пользой, не спам)
- Создание инструментов и калькуляторов, на которые ссылаются
- PR-публикации и пресс-релизы

**Важно:** наращивайте ссылки постепенно. Резкий рост выглядит неестественно и может привести к санкциям.

Подробнее: [Внешние ссылки и ссылочное продвижение](/blog/vneshnie-ssylki-i-ssylochnoe-prodvizhenie/).

## Шаг 8: Отслеживайте результаты

Проверяйте прогресс минимум раз в месяц:

| Метрика | Где смотреть | Что анализировать |
|---------|-------------|-------------------|
| Позиции | Topvisor, SE Ranking | Динамика по ключевым словам |
| Трафик | Яндекс.Метрика | Органический трафик по месяцам |
| Индексация | Вебмастер, Search Console | Сколько страниц в индексе |
| CTR | Search Console | Кликабельность сниппетов |
| Поведение | Метрика | Время на сайте, глубина просмотра |

**Что делать с данными:**
- Позиции растут → продолжать в том же темпе
- Позиции падают → проверить технические ошибки и конкурентов
- Трафик есть, но нет конверсий → проблема в юзабилити, а не в SEO
- CTR низкий → переписать title и description

## Шаг 9: Оптимизируйте и повторяйте

SEO — это цикл:

1. Аудит → 2. Ядро → 3. Контент → 4. Ссылки → 5. Анализ → повтор

Каждый цикл вы:
- Находите новые ключевые слова
- Создаёте новый контент
- Улучшаете существующие страницы
- Наращиваете авторитетность
