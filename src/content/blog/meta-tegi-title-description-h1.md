Мета-теги — это элементы HTML, которые сообщают поисковым системам информацию о странице. Title и description формируют сниппет в выдаче — именно его видит пользователь, когда решает, кликнуть или пролистать.

Правильные мета-теги = больше кликов из выдачи = больше трафика.

## Title (заголовок страницы)

Title — самый важный элемент для SEO. Он:

- Отображается как заголовок сниппета в выдаче
- Показывается во вкладке браузера
- Один из главных факторов ранжирования

### Правила написания title

| Правило | Рекомендация |
|---------|-------------|
| Длина | 50–60 символов (Яндекс обрезает после ~60) |
| Ключевое слово | В начале или ближе к началу |
| Уникальность | Каждая страница — свой title |
| Читаемость | Нормальное предложение, не набор ключей |
| Бренд | Название компании — в конце через « — » |

### Примеры

**Плохо:**
```
Купить, заказать SEO продвижение сайта в Москве — цена, стоимость SEO
```
Проблемы: переспам ключами, нечитаемо, Яндекс может подменить.

**Хорошо:**
```
SEO-продвижение сайта в Москве — от 30 000 ₽ | WebStudio
```
Почему: ключ в начале, есть конкретика (цена), бренд, читается как заголовок.

**Ещё примеры хороших title:**

- `Как создать сайт самостоятельно: пошаговая инструкция`
- `10 лучших хостингов для сайта в 2025 году — сравнение`
- `Внутренняя оптимизация сайта: полный чек-лист`

### Шаблоны title

| Тип страницы | Шаблон |
|-------------|--------|
| Статья-руководство | [Ключ]: пошаговая инструкция на [год] |
| Списочная статья | [N] лучших [предмет] в [год] — сравнение |
| Коммерческая | [Услуга] в [городе] — от [цена] ₽ | [Бренд] |
| Сравнение | [A] vs [B]: что выбрать в [год] |

### Частые ошибки в title

1. **Одинаковые title** на разных страницах — поисковик не понимает, какую показывать
2. **Слишком длинный** — обрезается, теряется смысл
3. **Без ключевого слова** — страница не ранжируется по запросу
4. **Кликбейт** — не соответствует содержанию → высокий отказ → падение позиций
5. **Title = H1** — потеря возможности использовать две разные формулировки

## Description (мета-описание)

Description — описание страницы, которое отображается под заголовком в сниппете.

**Важно:** description не влияет на ранжирование напрямую. Но влияет на CTR — а CTR влияет на позиции.

### Правила написания description

| Правило | Рекомендация |
|---------|-------------|
| Длина | 120–160 символов |
| Ключевое слово | Включить (выделяется жирным в выдаче) |
| Конкретика | Цифры, факты, выгода |
| Призыв | Мотивация кликнуть |
| Уникальность | Каждая страница — свой description |

### Примеры

**Плохо:**
```
Наша компания предлагает услуги SEO-продвижения. Мы работаем с 2010 года. Обращайтесь к нам за помощью.
```
Проблемы: ни о чём, нет конкретики, не мотивирует кликнуть.

**Хорошо:**
```
Как продвинуть сайт в Яндексе самостоятельно: 10 шагов от аудита до первого трафика. Бесплатные инструменты и чек-лист.
```
Почему: ключ есть, конкретика (10 шагов), выгода (бесплатно), понятно, что внутри.

### Что делать, если Яндекс/Google подменяет description

Поисковики часто генерируют описание сами, игнорируя ваш description. Это происходит, когда:

- Description не релевантен запросу
- Слишком короткий или общий
- Лучший фрагмент нашёлся в тексте страницы

**Как бороться:** пишите конкретные, информативные description, которые точно отвечают на запрос.

## H1 (заголовок первого уровня)

H1 — основной заголовок на странице. Именно его видит пользователь первым.

### Правила H1

| Правило | Рекомендация |
|---------|-------------|
| Количество | Ровно один H1 на страницу |
| Ключевое слово | Содержит основной ключ |
| Отличие от title | Не дублирует title дословно |
| Читаемость | Понятный заголовок, не SEO-спам |
| Длина | Нет жёсткого ограничения, но 40–70 символов оптимально |

### Связь title и H1

Title и H1 должны быть похожи по смыслу, но не одинаковы:

| Title | H1 |
|-------|-----|
| SEO-оптимизация сайта: полное руководство на 2025 год | SEO-оптимизация сайта: полное руководство |
| Как продвинуть сайт в Яндексе — пошаговый план | Как продвинуть сайт в Яндексе в 2025 году |
| 10 лучших хостингов 2025 — сравнение и цены | Лучшие хостинги для сайта: рейтинг 2025 |

**Зачем разделять:** две формулировки → две возможности использовать разные вариации ключа.

## Связь title, description и H1

Все три элемента должны работать вместе:

```
Title: Как подобрать ключевые слова для сайта: пошаговый гайд
Description: Как собрать семантическое ядро: Wordstat, конкуренты, кластеризация. 
             Пошаговая инструкция с примерами и бесплатными инструментами.
H1: Как подобрать ключевые слова для сайта
```

**Title** привлекает внимание в выдаче → **Description** убеждает кликнуть → **H1** подтверждает, что пользователь попал на нужную страницу.

## Чек-лист мета-тегов

### Title
- [ ] 50–60 символов
- [ ] Содержит основной ключ
- [ ] Уникальный для каждой страницы
- [ ] Читается как нормальный заголовок
- [ ] Не совпадает с H1

### Description
- [ ] 120–160 символов
- [ ] Содержит ключевое слово
- [ ] Конкретика (цифры, факты)
- [ ] Мотивирует кликнуть
- [ ] Уникальный для каждой страницы

### H1
- [ ] Один на страницу
- [ ] Содержит основной ключ
- [ ] Не дублирует title
- [ ] Понятный, читаемый

## Заключение

Мета-теги — это минимум усилий с максимальным эффектом. Правильный title может увеличить CTR в 2 раза, а это — прямой рост трафика без изменения позиций.

Пройдитесь по всем страницам сайта и проверьте: уникальные title, информативные description, один H1 с ключом. Это базовая гигиена SEO.
