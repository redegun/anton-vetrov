### Midjourney v7: революционные возможности

**Character Reference (--cref)**
Создание консистентных персонажей через серию изображений:
```
/imagine a business woman in a modern office --cref https://example.com/character.jpg
```

**Style Reference (--sref)**  
Применение художественного стиля с другого изображения:
```
/imagine city landscape --sref https://example.com/style.jpg
```

**Enhanced Text Rendering**
Качественная генерация текста внутри изображений:
```
/imagine logo design "COMPANY NAME" modern typography --text
```

**Video Generation (beta)**
Создание 4-секундных видеороликов из статичных изображений:
```
/imagine dancing in the rain --video --duration 4s
```

## Профессиональные техники промптинга

### Структура эффективного промпта

**Базовая формула:**
```
[Объект] + [Действие/поза] + [Окружение] + [Настроение/стиль] + [Технические параметры]
```

**Пример:**
```
/imagine confident CEO giving presentation, modern glass office, warm natural lighting, photojournalistic style --ar 16:9 --style raw
```

### Продвинутые техники

#### Multi-prompting (весовые коэффициенты)

```
/imagine sleek sports car::2 mountain landscape::1 golden hour lighting::1.5
```

#### Negative prompting

```
/imagine modern interior design --no furniture, cluttered, dark
```

#### Aspect ratio оптимизация

```
--ar 1:1    # Instagram post
--ar 16:9   # YouTube thumbnail  
--ar 9:16   # Stories/Reels
--ar 4:5    # Facebook post
--ar 3:2    # Печатные фотографии
```

#### Chaos и Stylize параметры

```
--chaos 25     # Добавляет вариативности (0-100)
--stylize 150  # Уровень художественности (0-1000)  
--quality 2    # Качество генерации (.25, .5, 1, 2)
```

### Отраслевые промпт-шаблоны

#### Для e-commerce

```
/imagine product photography, [product] on white background, studio lighting, clean commercial style, high resolution --ar 1:1 --style raw
```

#### Для недвижимости

```
/imagine luxurious [room type], modern interior design, warm lighting, architectural photography, high-end real estate --ar 3:2 --style raw
```

#### Для социальных сетей

```
/imagine lifestyle photography, [activity], natural lighting, authentic moment, social media style --ar 4:5 --stylize 200
```

#### Для презентаций

```
/imagine business concept illustration, [concept], clean modern style, infographic elements, professional --ar 16:9
```
