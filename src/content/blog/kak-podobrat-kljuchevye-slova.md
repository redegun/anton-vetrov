Ключевые слова — основа SEO-продвижения. Без них невозможно понять, что ищут ваши потенциальные клиенты и какой контент создавать. Правильно подобранные ключи приводят целевой трафик, неправильные — пустые просмотры.

В этой статье — полный процесс: от первичного сбора до кластеризации и распределения по страницам.

## Что такое ключевые слова и зачем они нужны

Ключевое слово (keyword) — это запрос, который пользователь вводит в поисковую строку. Например: «как создать сайт», «купить ноутбук Москва», «рецепт борща».

**Зачем подбирать ключевые слова:**

- Понять, что ищет ваша аудитория
- Создавать контент, который отвечает на реальные запросы
- Оценить конкуренцию и потенциал трафика
- Планировать структуру сайта

Без семантического ядра SEO превращается в угадывание.

## Типы ключевых слов

| Тип | Пример | Частотность | Конкуренция |
|-----|--------|-------------|-------------|
| Высокочастотные (ВЧ) | SEO оптимизация | 1,000+ | Высокая |
| Среднечастотные (СЧ) | SEO оптимизация сайта самостоятельно | 100–1,000 | Средняя |
| Низкочастотные (НЧ) | как оптимизировать сайт для Яндекса бесплатно | < 100 | Низкая |

**Стратегия для нового сайта:** начинать с НЧ и СЧ запросов. На ВЧ слишком высокая конкуренция — молодой сайт туда не пробьётся.

**Стратегия для сайта с трафиком:** добавлять ВЧ запросы и укреплять позиции по СЧ.

## Шаг 1: Мозговой штурм

Начните с простого — выпишите все запросы, которые могут искать ваши клиенты:

- Что они хотят купить / узнать / сделать?
- Какие проблемы решают?
- Какие термины используют (профессиональные и бытовые)?
- Как называют ваш продукт / услугу?

**Пример для SEO-агентства:**
- сео продвижение
- раскрутка сайта
- как попасть в топ яндекса
- стоимость продвижения сайта
- seo аудит

Не фильтруйте на этом этапе — записывайте всё.

## Шаг 2: Яндекс.Wordstat

Wordstat — основной инструмент для работы с русскоязычной семантикой. Показывает, сколько раз в месяц искали конкретный запрос.

**Как пользоваться:**

1. Перейдите на [wordstat.yandex.ru](https://wordstat.yandex.ru)
2. Введите базовый запрос
3. Смотрите левую колонку — вариации вашего запроса с частотностью
4. Правая колонка — похожие запросы (могут подкинуть идеи)

**Важные операторы:**

| Оператор | Что делает | Пример |
|----------|-----------|--------|
| «» (кавычки) | Точная фраза | «купить ноутбук» — только этот запрос |
| ! (восклицательный знак) | Фиксирует словоформу | !купить !ноутбук |
| + (плюс) | Учитывает предлоги | +как +сделать сайт |
| - (минус) | Исключает слово | ноутбук -бу -ремонт |

**Совет:** всегда проверяйте частотность в кавычках с восклицательным знаком — это реальная частота. Без кавычек Wordstat показывает суммарную частоту всех вариаций, что вводит в заблуждение.

## Шаг 3: Анализ конкурентов

Посмотрите, по каким запросам ранжируются ваши конкуренты. Это самый быстрый способ найти рабочие ключевые слова.

**Инструменты:**

- **Keys.so** — российский сервис, хорошо работает с Яндексом
- **Serpstat** — показывает ключи конкурентов и их позиции
- **Ahrefs** — мощный инструмент для анализа любого домена
- **SpyWords** — специализируется на конкурентном анализе в Рунете

**Что делать:**
1. Введите домен конкурента
2. Выгрузите список ключевых слов, по которым он ранжируется
3. Отфильтруйте по вашей тематике
4. Добавьте в своё ядро те, которых у вас нет

## Шаг 4: Расширение ядра

Дополнительные источники ключевых слов:

- **Подсказки Яндекса и Google** — начните вводить запрос и смотрите варианты
- **Блок «Вместе с этим ищут»** — внизу выдачи
- **Google Keyword Planner** — бесплатный инструмент в Google Ads
- **Answer the Public** — визуализирует вопросы по теме
- **Форумы и сообщества** — как люди формулируют свои проблемы
- **Отзывы и FAQ** — реальные вопросы клиентов

## Шаг 5: Чистка ядра

После сбора у вас будет список из сотен или тысяч ключевых слов. Нужно убрать лишнее:

- **Нерелевантные** — запросы не по вашей теме
- **Информационные** (если у вас коммерческий сайт без блога) — «что такое SEO» не приведёт к продаже
- **Нулевые** — частотность 0 в кавычках
- **Дубли** — одинаковые по смыслу запросы
- **Геозависимые не из вашего региона**

## Шаг 6: Кластеризация

Кластеризация — группировка ключевых слов по смыслу. Один кластер = одна страница на сайте.

**Принцип:** если по двум запросам в топ-10 выдачи попадают одни и те же страницы — эти запросы можно продвигать на одной странице.

**Методы кластеризации:**

1. **Ручная** — подходит для небольших ядер (до 200 ключей). Группируете по смыслу вручную.
2. **Автоматическая** — сервисы Just-Magic, Serpstat, Rush Analytics. Анализируют выдачу и группируют по пересечению топа.

**Пример кластера:**

```
Кластер: «SEO-аудит»
- seo аудит сайта
- как провести сео аудит
- чек-лист сео аудита
- аудит сайта для продвижения
→ Создаём одну страницу: /blog/seo-audit-sajta/
```

## Шаг 7: Распределение по страницам

Каждый кластер привязывается к конкретной странице:

- **Коммерческие кластеры** → страницы услуг / товаров
- **Информационные кластеры** → статьи в блоге
- **Навигационные кластеры** → категории, разделы

**Правила:**
- Один кластер — одна страница (не размазывайте один ключ по нескольким страницам)
- Основной ключ кластера → в title, H1 и первый абзац
- Дополнительные ключи → в H2, H3 и текст
- LSI-слова (тематические) — распределите естественно по тексту

## Ошибки при подборе ключевых слов

1. **Ориентироваться только на ВЧ** — высокая конкуренция, низкий шанс попасть в топ
2. **Игнорировать интент** — «яблоко» может быть фруктом или брендом
3. **Не проверять в кавычках** — реальная частота может быть в 10 раз ниже
4. **Один ключ на весь сайт** — сайт должен отвечать на десятки и сотни запросов
5. **Забывать обновлять ядро** — тренды меняются, появляются новые запросы

## Заключение

Подбор ключевых слов — это не разовая задача, а постоянный процесс. Начните с мозгового штурма, расширьте через Wordstat и конкурентов, кластеризуйте и распределите по страницам.

Главное — отталкивайтесь от реальных запросов пользователей, а не от своих предположений. Wordstat не врёт.
