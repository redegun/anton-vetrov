Яндекс — основная поисковая система в России. Его доля в рунете — около 60%. Продвижение в Яндексе отличается от Google: здесь свои факторы ранжирования, свои инструменты и свои правила.

В этой статье — всё, что нужно знать для продвижения сайта в Яндексе в 2025 году.

## Чем Яндекс отличается от Google

| Фактор | Яндекс | Google |
|--------|--------|--------|
| Коммерческие факторы | Очень важны | Менее выражены |
| Региональность | Сильная привязка | Слабая |
| Ссылки | Меньше веса | Больше веса |
| Поведенческие факторы | Один из главных сигналов | Важны, но меньше |
| ИКС | Есть | Нет аналога |
| Турбо-страницы | Есть | AMP (почти мёртв) |
| Нейросети в выдаче | YandexGPT | AI Overviews |

**Главный вывод:** в Яндексе важнее качество сайта и поведение пользователей, чем ссылочная масса.

## Коммерческие факторы ранжирования

Для коммерческих сайтов (услуги, магазины) Яндекс оценивает «коммерческие факторы» — сигналы доверия и удобства.

**Что проверяет Яндекс:**

- **Контакты** — телефон, email, адрес, карта проезда
- **Юридическая информация** — ИНН, ОГРН, реквизиты
- **Цены** — должны быть указаны (не «цена по запросу»)
- **Доставка и оплата** — подробные условия
- **Ассортимент** — достаточный для ниши
- **Отзывы** — реальные отзывы клиентов
- **Гарантии** — условия возврата, гарантийное обслуживание
- **О компании** — история, команда, сертификаты

**Частая ошибка:** сайт с хорошим контентом, но без контактов и цен — Яндекс не покажет его в коммерческой выдаче.

## Региональность

Яндекс привязывает сайт к конкретному региону. Запрос «ремонт квартир» в Москве и Новосибирске покажет разные результаты.

**Как настроить регион:**

1. **Яндекс.Бизнес** — зарегистрируйте организацию с точным адресом
2. **Яндекс.Вебмастер** — укажите регион сайта в настройках
3. **Контакты на сайте** — адрес, телефон с кодом города
4. **Поддомены / подпапки** — для мультирегиональных сайтов

**Для информационных сайтов** региональность менее важна — статьи показываются вне зависимости от региона.

## ИКС (Индекс Качества Сайта)

ИКС — комплексная метрика Яндекса, которая оценивает «качество» сайта. Видна в Вебмастере.

**Что влияет на ИКС:**
- Трафик сайта
- Удовлетворённость пользователей (поведенческие факторы)
- Полнота и качество контента
- Техническое состояние

**Как повысить ИКС:**
- Увеличивать трафик (больше качественных страниц)
- Улучшать юзабилити (меньше отказов, больше глубина просмотра)
- Обновлять контент
- Исправлять технические ошибки

ИКС растёт медленно. Не пытайтесь его «накрутить» — это приведёт к санкциям.

## Поведенческие факторы

Яндекс активно использует данные о поведении пользователей для ранжирования.

**Что отслеживает Яндекс:**

| Фактор | Что значит | Как улучшить |
|--------|-----------|-------------|
| CTR в выдаче | Кликабельность сниппета | Привлекательный title и description |
| Показатель отказов | % ушедших сразу | Релевантный контент, быстрая загрузка |
| Время на сайте | Сколько пользователь провёл | Полезный контент, видео, интерактив |
| Глубина просмотра | Сколько страниц посмотрел | Перелинковка, рекомендации |
| Возврат в выдачу | Вернулся ли пользователь искать дальше | Полнота ответа на запрос |

**Важно:** накрутка поведенческих факторов (ботами) — верный путь к бану. Яндекс научился это обнаруживать и жёстко наказывает.

## Яндекс.Вебмастер

Яндекс.Вебмастер — основной инструмент для работы с Яндексом. Подключите его обязательно.

**Что делать в Вебмастере:**

1. Подтвердить права на сайт
2. Добавить sitemap.xml
3. Указать регион
4. Проверить индексацию (все ли страницы в индексе)
5. Мониторить ошибки (404, 500, проблемы с robots.txt)
6. Настроить уведомления о санкциях

**Полезные разделы:**
- «Диагностика» — технические ошибки
- «Поисковые запросы» — по каким ключам показывается сайт
- «Ссылки» — внешние и внутренние ссылки
- «Безопасность» — проверка на вирусы и фишинг

## Яндекс.Бизнес и Яндекс.Карты

Для локального бизнеса регистрация в Яндекс.Бизнесе обязательна:

- Сайт появится на Яндекс.Картах
- Привязка к региону усилит ранжирование
- Отзывы на картах влияют на доверие
- Можно добавить фото, меню, прайс-лист

**Как оптимизировать карточку:**
- Заполните все поля (категория, часы работы, услуги)
- Добавьте качественные фото
- Попросите клиентов оставить отзывы
- Отвечайте на отзывы (даже негативные)

## Контент для Яндекса

Яндекс в 2025 году оценивает контент через нейросети. Ключевые принципы:

1. **Полнота** — раскройте тему полностью, не оставляйте вопросов
2. **Экспертность** — покажите опыт, приведите данные
3. **Уникальность** — не копируйте, не рерайтите
4. **Актуальность** — обновляйте устаревшую информацию
5. **Естественность** — пишите для людей, не для робота

**Яндекс наказывает:**
- SEO-тексты «ни о чём» (водянистые, без конкретики)
- Переоптимизацию (слишком много ключей)
- Скрытый текст
- Дорвеи (страницы только для поисковика)

## Турбо-страницы

Турбо-страницы — ускоренная версия ваших страниц, которые загружаются мгновенно в мобильном Яндексе.

**Плюсы:**
- Мгновенная загрузка
- Специальная метка в выдаче
- Лучшие позиции на мобильном

**Минусы:**
- Ограниченный дизайн
- Нельзя использовать произвольный JavaScript
- Сложнее отслеживать конверсии

**Кому нужны:** медиа, блоги, новостные сайты. Для сложных коммерческих сайтов — обычно не стоит.

## Пошаговый план продвижения в Яндексе

1. **Зарегистрируйтесь** в Яндекс.Вебмастере и Яндекс.Бизнесе
2. **Проведите аудит** — технические ошибки, индексация, скорость
3. **Настройте регион** — через Вебмастер и карточку организации
4. **Заполните коммерческие факторы** — контакты, цены, доставка
5. **Соберите семантику** — через Яндекс.Wordstat
6. **Создайте контент** — под ключевые запросы, лучше конкурентов
7. **Настройте перелинковку** — свяжите тематически близкие страницы
8. **Работайте с отзывами** — собирайте и отвечайте
9. **Мониторьте** — позиции, трафик, ошибки в Вебмастере
10. **Обновляйте** — контент, цены, информацию о компании
