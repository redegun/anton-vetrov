# Базы знаний antonvetrov.ru

## Структура

### 1. `knowledge/` — Личные знания Антона
Мысли, опыт, мнения, личные заметки. Уникальный контент от первого лица.

**Разделы:**
- `tech/` — мнения о технологиях, CMS, фреймворках
- `dev/` — подходы к разработке, принципы
- `seo/` — наблюдения по SEO
- `business/` — фриланс, клиенты, ценообразование
- `notes/` — разные заметки и идеи

**Используется для:** статей с личной экспертизой, авторским мнением.

### 2. `knowledge-base/` — Общие знания (внешние источники)
Конспекты из видео, статей, книг, подкастов. Экспертные знания из интернета.

**Структура:** организуется по темам, каждый файл — выжимка из источника с указанием откуда взято.

**Используется для:** услуг, экспертных статей, обоснования подходов.

## Правила

1. **Личные мысли** → `knowledge/`
2. **Из внешних источников** (видео, статьи, книги) → `knowledge-base/`
3. Каждый файл — `.md`, с понятным именем
4. В `knowledge-base/` всегда указывать источник (ссылка, автор, дата)
