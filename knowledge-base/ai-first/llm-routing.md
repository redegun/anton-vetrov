# AI First — Маршрутизация LLM

**Источник:** https://youtu.be/yWAJ6ICJL4c

## Почему не одна LLM

- Разные rate-limits
- Разная цена за миллион токенов
- Сильные / слабые модели
- Быстрые / медленные модели

## Решение

Ядро системы осуществляет **выбор и ротацию LLM** — запрос маршрутизируется в подходящую модель в зависимости от задачи, нагрузки и стоимости.
