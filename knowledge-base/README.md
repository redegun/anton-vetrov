# Knowledge Base — Общие знания

Экспертные знания из внешних источников: видео, статьи, книги, подкасты.

Структура организуется по темам. Каждый файл — конспект/выжимка из источника с указанием откуда взято.

## Отличие от knowledge/

- **knowledge/** — личные мысли, опыт и заметки Антона
- **knowledge-base/** — общие знания из внешних источников (для экспертизы)
